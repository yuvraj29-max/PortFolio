## 💾 Local Quickstart

### Prerequisites
You only need a modern web browser to open the application.

### Execution
1. Clone this repository:
   ```bash
   git clone https://github.com/yuvraj29-max/PortFolio.git
   ```
2. Navigate into the project folder:
   ```bash
   cd PortFolio
   ```
3. Start a local server (e.g., via Python):
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to:
   ```url
   http://localhost:8000
   ```
